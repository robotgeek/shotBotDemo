 *  Pin 8 - RobotGeek Pushbutton relay/pumping station 2
 *
 *  Control Behavior:
 *    Push each button once to activate the coresponding pump for a time.
 *    Alter the defines  PUMP_1_TIME and PUMP_2_TIME to change the time that each
